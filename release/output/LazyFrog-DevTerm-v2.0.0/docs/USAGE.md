# Usage

LazyFrog DevTerm is designed for quick navigation and short, repeatable actions. Everything is keyboard-first and centered around common developer routines.

---

## Main menu

- Use **Up/Down** to move
- Press **Enter** to open a module
- Press **1–4** to jump directly to a module
- Press **Q** to quit

---

## GitHub Scanner

**Why it helps:** it’s a fast way to sanity-check repos or gather references without leaving your terminal.

- **S**: search
- **L**: language filter
- **W**: add to watchlist
- **V**: view watchlist
- **J**: save JSON to results/
- **M**: save Markdown to results/
- **Esc**: back

**Tip:** use quotes for exact phrases, and combine keywords with a language filter for better results.

---

## Task Runner

**Why it helps:** turns your most common commands into named actions with consistent results.

- **Enter/X**: run task
- **A**: add task
- **E**: edit task
- **D**: delete task
- **H**: history

**Important:** Git tasks must be run inside a Git repository folder. If you see exit code 128, start LazyFrog from a repo directory.

---

## System Monitor

**Why it helps:** quick health checks without opening full system tools.

- Live updates at the refresh interval
- **R**: refresh
- **S**: save snapshot
- **I**: details
- **Esc**: back

---

## Help

- Use **Up/Down** and **Enter** to open a topic
- **Esc** returns to the previous screen
